# F8 — AI Style Background Generation

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an "AI Style" button in the layout selector. Claude Sonnet generates a **no-text editorial background** HTML composition from the product photo → Puppeteer renders it to a background PNG → saved to disk. The user then layers the headline (from the copy pool) and free text boxes (F7) on top. Background is frozen after generation; text is fully editable. Export bakes everything together via the existing reposition route.

---

## Key Design Decisions

| Decision | Choice | Reason |
|----------|--------|--------|
| Background storage | PNG file on disk (in uploads dir) | Avoids storing raw HTML in DB; reuses existing file infrastructure |
| Background reference in AdSpec | `aiBgImagePath?: string` | Minimal AdSpec change; file path is stable |
| Template render | Reads background PNG (not product image) | Template gets bg as `imageBase64`; headline + text boxes injected on top as usual |
| Headline | From copy pool, same as other layouts | User can reposition, change, or hide (F7 hideHeadline) |
| Format switch | Re-generate (new AI call) | Background is format-specific; can't cheaply crop a composition |
| Language switch | OK — only copy changes, not background | Standard reposition route; no new AI call |

---

## Architecture

```
[AI Style button click]
        │
        ▼
POST /api/ai-style
  ├── Read product image from storage
  ├── Call generateAIBackground(imageBase64, format) → Claude Sonnet → background HTML
  ├── Puppeteer: setContent(bgHtml) → screenshot → background PNG
  ├── Save background PNG to uploads dir → bgPath
  ├── Pick headline from copy pool (same as other layouts)
  ├── Create AdSpec { templateId: "ai_background", aiBgImagePath: bgPath, ... }
  └── Return RenderResult (pngUrl = composite of bg + headline)

[LiveAdCanvas] — shows bg + headline overlay
  └── User drags headline, adds text boxes (F7), hides headline → all local state

[Approve button]
  └── POST /api/reposition with textBoxes + hideHeadline → bakes all layers into final PNG
```

## Task 2: Create AI background prompt

**Files:** `lib/ai/aiBackground.ts` (new)

- [ ] **Step 1: Create the file**

```typescript
import Anthropic from "@anthropic-ai/sdk";
import { withRetry } from "@/lib/ai/retry";
import type { Format } from "@/lib/types";
import { FORMAT_DIMS } from "@/lib/types";

const MODEL = "claude-sonnet-4-6";

export async function generateAIBackground(
  imageBase64: string,
  mimeType: "image/jpeg" | "image/png" | "image/webp",
  format: Format,
): Promise<string> {
  const apiKey = process.env.ANTHROPIC_API_KEY;
  if (!apiKey) throw new Error("ANTHROPIC_API_KEY not set");

  const { w, h } = FORMAT_DIMS[format];
  const client = new Anthropic({ apiKey });

  const response = await withRetry(() => client.messages.create({
    model: MODEL,
    max_tokens: 4000,
    messages: [{
      role: "user",
      content: [
        {
          type: "image",
          source: { type: "base64", media_type: mimeType, data: imageBase64 },
        },
        {
          type: "text",
          text: `You are a world-class art director for luxury product advertising. Create a rich editorial background composition featuring this product.

CRITICAL — DO NOT include:
- Any text, letters, words, numbers, or labels
- Buttons, icons, UI elements, price tags
- Placeholder text or lorem ipsum
- Watermarks or credits

COMPOSITION:
- Canvas: ${w}px × ${h}px (width × height)
- Feature the product prominently — it should be the visual hero
- Add editorial depth: subtle gradients, color washes, geometric shapes, framing elements
- Sophisticated palette — max 3 colors. Think Vogue, Harper's Bazaar editorial spreads
- The product image must be embedded directly in the HTML as a base64 data URI
- Use the exact dimensions — no scrollbars, no overflow

STYLE OPTIONS (pick one that best suits the product):
- Full-bleed product over a solid color panel (split or layered)
- Product centered with a lush gradient background
- Editorial split: solid geometric block + product panel
- Soft vignette or color wash over full-bleed product
- Geometric frame elements around the product

Return ONLY a complete, self-contained HTML page. No markdown, no explanation, no code blocks.
The page <body> must be exactly ${w}px × ${h}px. Use position:absolute for all elements.
Embed the product image using: <img src="data:${mimeType};base64,REPLACE_WITH_IMAGE_DATA" ...>
Replace REPLACE_WITH_IMAGE_DATA with the actual base64 string of the product image provided above.`,
        },
      ],
    }],
  }), "generateAIBackground");

  const html = (response.content[0] as { type: "text"; text: string }).text.trim();

  // Strip markdown code fences if Claude wraps the response
  const clean = html.replace(/^```html?\n?/i, "").replace(/\n?```$/, "").trim();
  return clean;
}
```

## Task 3: Create the API route

**Files:** `app/api/ai-style/route.ts` (new)

- [ ] **Step 1: Create the route**

```typescript
import { NextRequest, NextResponse } from "next/server";
import { newId } from "@/lib/ids";
import { getDb } from "@/lib/db";
import { read as readStorage } from "@/lib/storage";
import { generateAIBackground } from "@/lib/ai/aiBackground";
import { renderAd } from "@/lib/render/renderAd";
import { FORMAT_DIMS } from "@/lib/types";
import type { Format, Language, AdSpec } from "@/lib/types";
import path from "path";
import { writeFile } from "fs/promises";
import sharp from "sharp";

export async function POST(req: NextRequest) {
  try {
    const { imageId, lang = "en", format = "4:5" } = await req.json() as {
      imageId: string;
      lang?: Language;
      format?: Format;
    };

    if (!imageId) return NextResponse.json({ error: "imageId required" }, { status: 400 });

    const db = await getDb();

    // 1. Load product image
    const imageRow = db.prepare("SELECT url, width, height FROM images WHERE id = ?").get(imageId) as
      { url: string; width: number; height: number } | undefined;
    if (!imageRow) return NextResponse.json({ error: "Image not found" }, { status: 404 });

    const rawBuffer = await readStorage("uploads", imageRow.url);
    const mimeType: "image/jpeg" | "image/png" | "image/webp" =
      imageRow.url.match(/\.png$/i) ? "image/png"
      : imageRow.url.match(/\.webp$/i) ? "image/webp"
      : "image/jpeg";
    const { w, h } = FORMAT_DIMS[format as Format];

    // Resize product image to match the target canvas (Claude needs it at the right scale)
    const resized = await sharp(rawBuffer).resize(w, h, { fit: "cover" }).toBuffer();
    const imageBase64 = resized.toString("base64");

    // 2. Generate background HTML via Claude Sonnet
    console.log("[ai-style] Generating background HTML...");
    const backgroundHtml = await generateAIBackground(imageBase64, mimeType, format as Format);

    // 3. Render background HTML via Puppeteer → background PNG
    // We use a minimal AdSpec with a throwaway render to just screenshot the HTML
    const bgId = newId();
    const bgFilename = `bg_${bgId}.png`;
    const uploadsDir = path.join(process.cwd(), "uploads");
    const bgPath = path.join(uploadsDir, bgFilename);

    // Render the background HTML using a temporary page (reuse browser from renderAd)
    const { getBrowserForExport } = await import("@/lib/render/renderAd");
    const browser = await getBrowserForExport();
    const page = await browser.newPage();
    try {
      await page.setViewport({ width: w, height: h });
      await page.setContent(backgroundHtml, { waitUntil: "networkidle0", timeout: 30000 });
      const pngBuffer = await page.screenshot({ type: "png" }) as Buffer;
      await writeFile(bgPath, pngBuffer);
    } finally {
      await page.close();
    }
    console.log(`[ai-style] Background PNG saved: ${bgFilename}`);

    // 4. Get or generate a headline from the copy pool
    const copyPoolRow = db.prepare("SELECT data FROM copy_pools WHERE image_id = ?").get(imageId) as
      { data: string } | undefined;
    let headline = "";
    let primarySlotId = "";
    if (copyPoolRow) {
      const pool = JSON.parse(copyPoolRow.data) as { slots: Array<{ id: string; lang: string; slotType: string; text: string; angle?: string }> };
      const slot = pool.slots.find(s => s.lang === lang && s.slotType === "headline")
        ?? pool.slots.find(s => s.slotType === "headline");
      if (slot) { headline = slot.text; primarySlotId = slot.id; }
    }

    // 5. Create AdSpec
    const adSpecId = newId();
    const adSpec: AdSpec = {
      id: adSpecId,
      imageId,
      format: format as Format,
      lang: lang as Language,
      familyId: "ai",
      templateId: "ai_background",
      zoneId: "A",
      primarySlotId,
      copy: { headline },
      theme: {
        fontHeadline: "Playfair Display",
        fontSize: 48,
        color: "#ffffff",
        bg: "#000000",
        radius: 0,
        shadow: false,
      },
      renderMeta: { w, h },
      aiBgImagePath: bgPath,
    };

    // 6. Render the final ad (background + headline via renderAd)
    const safeZonesRow = db.prepare("SELECT data FROM safe_zones WHERE image_id = ?").get(imageId) as
      { data: string } | undefined;
    const safeZones = safeZonesRow ? JSON.parse(safeZonesRow.data) : { zones: [], avoidRegions: [] };

    const { pngUrl, renderResultId } = await renderAd(adSpec, safeZones);

    // 7. Persist AdSpec and RenderResult
    db.prepare("INSERT INTO ad_specs (id, image_id, data) VALUES (?, ?, ?)").run(
      adSpecId, imageId, JSON.stringify(adSpec)
    );
    db.prepare(`INSERT INTO render_results
      (id, ad_spec_id, image_id, family_id, template_id, primary_slot_id, png_url, approved, replaced_by, created_at)
      VALUES (?, ?, ?, ?, ?, ?, ?, 0, NULL, datetime('now'))`
    ).run(renderResultId, adSpecId, imageId, "ai", "ai_background", primarySlotId, pngUrl);

    return NextResponse.json({
      ok: true,
      result: {
        id: renderResultId,
        adSpecId,
        imageId,
        familyId: "ai",
        templateId: "ai_background",
        primarySlotId,
        format,
        pngUrl,
        approved: false,
        createdAt: new Date().toISOString(),
        headlineText: headline,
        lang,
      },
    });
  } catch (err) {
    console.error("[ai-style] Error:", err);
    return NextResponse.json({ error: String(err) }, { status: 500 });
  }
}
```

> **Note on `getBrowserForExport`:** The current `renderAd.ts` does not export the browser getter. In Task 5 we'll export a helper function `getBrowserForExport()` that returns the cached browser instance, so the route can reuse it without launching a second Chromium.

```typescript
import { registerTemplate } from "./registry";

registerTemplate({
  id: "ai_background",
  familyId: "ai",
  name: "AI Style",
  supportedZones: ["A", "B", "C"],
  themeDefaults: {
    fontHeadline: "Playfair Display",
    fontSize: 48,
    color: "#ffffff",
    bg: "#000000",
    radius: 0,
    shadow: false,
  },
  maxLines: 2,
  copySlots: ["headline"],
  preferredHeadlineLength: "short",

  build(spec, imageBase64) {
    const { w, h } = spec.renderMeta;
    const headline = spec.copy.headline ?? "";
    const hideHeadline = spec.hideHeadline === true;

    // Headline position: default 65% down the canvas (editorial sweet spot)
    const headlineYPct = spec.headlineYOverride !== undefined
      ? spec.headlineYOverride * 100
      : 65;

    const headlineHtml = hideHeadline || !headline ? "" : `
      <p data-fit-headline style="
        position:absolute;
        left:0;right:0;
        top:${headlineYPct}%;
        transform:translateY(-50%);
        margin:0;
        padding:0 60px;
        font-family:'Playfair Display',serif;
        font-size:${spec.theme.fontSize}px;
        font-weight:700;
        color:${spec.theme.color};
        text-align:center;
        line-height:1.2;
        letter-spacing:0.02em;
        text-shadow:0 2px 12px rgba(0,0,0,0.5);
        word-break:normal;
        overflow-wrap:normal;
      ">${headline}</p>`;

    return `
      <div style="position:relative;width:${w}px;height:${h}px;overflow:hidden;background:#000;">
        <img
          src="data:image/png;base64,${imageBase64}"
          style="position:absolute;top:0;left:0;width:100%;height:100%;object-fit:cover;"
          alt=""
        />
        ${headlineHtml}
      </div>`;
  },
});
```

## Known limitations (v1 — acceptable)

- Format switch not supported for AI Style (requires new AI call — future enhancement)
- Background regeneration always makes a new Claude Sonnet call (no caching by imageId)
- Background PNG stored indefinitely on disk (same as product images — cleared only by user)
- AI Style not available for Surprise Me mode (separate creative path)
- `hideHeadline` only suppresses the template-level headline; `ai_background` template headline is separate from `aiSurprise.tsx` path
